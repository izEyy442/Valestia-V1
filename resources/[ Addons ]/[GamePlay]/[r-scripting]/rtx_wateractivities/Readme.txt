Install instructions (Standalone):
1. Put rtx_wateractivities folder to your resources
2. Open config.lua file
3. Configure your config.lua to your preferences
4. Put rtx_wateractivities to the server.cfg

Install instructions (QBCore):
1. Put rtx_wateractivities to your resources.
2. Open config.lua file
3. Replace Config.Framework = "standalone" with Config.Framework = "qbcore"
4. Configure your config.lua to your preferences
5. Put rtx_wateractivities to the server.cfg
6. Add new items to qb-core/shared/items.lua - items name: banana, inflatable, parasailing, ski, circle, bed1, bed2, bed3, bed4
